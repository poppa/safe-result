all

  • all<T>(res: Array<T | Promise<T>>): Promise<SuccessCollection<SuccessResult<ValueType<T>>> | FailureCollection<FailureResult>>
    • Defined in src/collection.ts:218

    Creates a Promise that resolved to either a SuccessCollection or a FailureCollection.

    This behaves pretty much the same as Promise.all(), so it will abort and return a FailureCollection as soon as a value is a rejected value.
